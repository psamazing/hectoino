가입방법

간편한 가입, 쉬운 이용! 서비스 가입 시 아래 내용을 참고하세요.

  • 01 서비스 가입 홈페이지 > 서비스가입/해지 > 서비스가입 탭 또는
    이용사이트 게재된 배너에 진입하여
    가입창에 휴대폰번호 입력 후 인증을 진행해주세요. (최초1회)

  • 02 앱설치 입력한 휴대폰 번호로 문자 수신 확인 >
    문자 내 앱 다운로드 URL 클릭 > 구글 플레이스토어/앱스토어에서
    앱을 설치    해 주세요.

  • 03 앱설정 앱 실행 > 비밀번호 입력 > 비밀번호 분실 시 사용할
    이메일 입력    을진행해 주세요. (최초1회)

  • 04 이용 사이트 ID/PW 등록 자주 이용하는 사이트 클릭 > 사이트 ID/PW 입력 >
    저장     버튼 클릭해 주세요.
